Sketckes

Project Ideation
For this project I want to create a book shelf, in which will have an organized set of books and on the screen two buttons will be display one for changing the color of the books and the other one, when the user click on it it will disorganized the books dragging them down the rectangle canvas display. And, after that the user will have the ability to reorganized the books by size or color.
